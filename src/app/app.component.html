Thanks for checking out my Angular App!
Some things to note:
  • It is a community store for sharing, selling, and buying items. If you like a product, you can message the seller to make an offer.
  • All the images are from Unsplash and are free to use. I only used a handful of them for this demo, so you may see the same image multiple times.
  • Check out the infinite scroll feature. I implemented it my self using the a HostListener. I initially load 8 items and then load more every time you scroll to the bottom of the page.
  • I used Bootstrap Sass for the styling.
  • The app uses a Nodejs backend with a MongoDB database. The backend is hosted on Render

I worked really hard on this app, so please let me know if you have any feedback or questions.
Take a look at the source code, if you are interested in how I built it. Github
